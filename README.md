# AI Mental Health Chatbot 

An AI-powered chatbot that detects emotions from user text and provides personalized mental wellness support.

## Features
- Detects user emotions from text using AI
- Provides personalized mental wellness responses
- Identifies crisis situations and shows emergency helplines
- Full admin monitoring panel

## Tech Stack
Python · Flask · HuggingFace Transformers · SQLite · HTML · CSS · JavaScript

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Open browser: `http://localhost:5000`
